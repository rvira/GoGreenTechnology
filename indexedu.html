Amazing Bluetooth Controlled Bot

Numerous applications can be made using the EDU-ROBO. One such approach is made using a bluetooth sensor. So here we have a user phone to control the bot movements using an app and the connection between the kit and user phone is made using bluetooth. This is a demonstration, using the Bluetooth module wherein the smartphone is used as a Remote Control Unit for controlling the robot functions. MIT App Inventor for making this Smartphone Application wherein the USER INTERFACE for remote control through Bluetooth has been made by dragging and dropping buttons, text boxes, etc. Then each button has been programmed with their respective functions using the block code method.
